# Extension Icons

Add the following icon files to this directory:

- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels) 
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

For development, you can use placeholder icons or create simple shield-themed icons.

The icons should represent:
- Security/protection (shield icon)
- Family safety
- Child protection

Color scheme: Blue and white (matching the Guardian Shield brand)
